Welcome To Churros

Introduction

Hi, This app is called Churros and it is made to plan out Events for users so that they are able to see what events are coming up so they don't miss it.

Navigation

In this App you will be able to navigate through the necessary pages through the options at the bottom of the page The pages that are there are the profile page and the Events page.

In the Profile page you will be able to see your information that you have entered at registration and the Image that you made your for your Profile picture.

From the Profile page you can navigate through to the events page at the bottom of the page and there will also be a menu on the top right of the page which will have an option to refresh page, go and add a new Event or you can Logout.

From the Events page there will be Events that will be shown whenever you add an Event. There will also be a plus button on the bottom right where you will be able to add events onto the list. Once you have done that you will also be able to swipe left on the event lists and you'll be able to send information about the event to your friends or family.

There is also an Add Events page where you will be able to add the information you need for the event and there will be a create button on the bottom and once pressed you will see a notification that an event is created and will redirect you back to the events page
